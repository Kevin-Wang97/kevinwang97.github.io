4.HTTP

4.1什么是http

传输控制协议，通常运行在TCP中，指定了客户端可能发来什么消息以及得到什么样的响应，

  • 文本：html,字符串
  • 超文本：图片，音乐，视频，定位，地图
  • https:安全的
  • 443

4.2，两个时代

4.3http请求，相应

客户端–发请求–服务器

服务器-响应–客户端

Request URL: https://www.baidu.com/ 请求地址
Request Method: GET get/方法/post/方法
Status Code: 200 OK 状态码：200 3 5 4
Remote Address: 61.135.169.125:443
Referrer Policy: no-referrer-when-downgrade

响应：

1.请求行

  • 请求行的请求方式：GET

    • 请求方式：Get,Post,HEAD,DELETE,PUT,…

      POST:安全不高效

      GET：高效不安全

2.消息头
3.响应状态码

200：请求响应成功 200

3xx:请求重定向

  • 重新到给你的新位置

4xx:找不到资源 404

5xx:服务器代码错误 500 502：网关错误
